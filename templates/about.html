Tentang
FishCount AI

Sistem pendeteksian dan penghitungan ikan otomatis berbasis deep learning YOLOv8, dikembangkan sebagai proyek akhir mata kuliah Pengolahan Citra Digital di Teknik Informatika, Universitas Maritim Raja Ali Haji.

Edition
🎯

Tujuan Sistem

Sistem ini dirancang untuk mendeteksi dan menghitung jumlah ikan secara otomatis pada keramba jaring apung menggunakan teknologi computer vision berbasis deep learning YOLOv8. Tujuannya adalah membantu pembudidaya ikan mendapatkan data populasi secara cepat, akurat, dan non-invasif.

🧠

Tech Stack

  • Python 3.11 Backend runtime
  • Flask Web framework (REST API)
  • YOLOv8 Object detection model
  • Ultralytics Training & inference engine
  • Pillow Image preprocessing
  • NumPy Komputasi statistik
🔄

Alur Kerja Sistem

  • 1Pengguna mengunggah gambar ikan via antarmuka web
  • 2Validasi & preprocessing citra dengan Pillow
  • 3Inferensi model YOLOv8 custom-trained
  • 4Ekstraksi bounding box, kelas, dan confidence
  • 5Komputasi density map dan statistik
  • 6Rendering hasil anotasi dan pelaporan otomatis
